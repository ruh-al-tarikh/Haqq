## 📖 About

**HAQ** is a digital archive and storytelling project connected to the YouTube channel:  
🎥 [Ruh-Al-Tarikh](https://www.youtube.com/@Ruh-Al-Tarikh)

This project is dedicated to reviving forgotten layers of Islamic history, prophetic stories, ancient wisdom, and unseen realities. Each episode is carefully crafted to:  
- Reveal hidden truths from the Quran  
- Explore historical and prophetic narratives  
- Reflect on signs of the past, present, and future
